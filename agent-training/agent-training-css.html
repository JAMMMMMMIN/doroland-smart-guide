📘 정보국 1급 기밀 (CSS 스타일 백과사전)

0. 선택자(Selector)의 비밀

CSS 요원은 HTML 요원이 만든 뼈대 중 누구에게 예술적 마법을 걸지 먼저 찾아야 해요. 이것을 '선택자'라고 부릅니다.

  • 태그 선택자: h1 { } 모든 h1 태그에 마법을 겁니다.
  • 클래스 선택자: .btn { } class="btn" 별명을 가진 모든 요소에 마법을 겁니다. (앞에 . 필수!)
  • 아이디 선택자: #logo { } id="logo"라는 유일한 이름을 가진 요소에 마법을 겁니다. (앞에 # 필수!)
  • 자식 선택자: div p { } div 안에 들어있는 p 태그만 골라냅니다.
  • 전체 선택자: * { } 모든 요소에 한꺼번에 마법을 겁니다.

1. 텍스트 꾸미기 마법 (Typography)

  • color: 글자 색상을 바꿉니다. (예: red, #ff0000, rgb(255,0,0))
  • font-size: 글자 크기를 조절합니다. (예: 16px, 1.2rem, 120%)
  • font-weight: 글자 두께를 정합니다. (예: normal, bold, 700)
  • font-family: 글꼴을 바꿉니다. (예: 'Pretendard', sans-serif)
  • text-align: 줄 안에서의 정렬 방향입니다. (예: left, center, right, justify)
  • text-decoration: 선으로 꾸밉니다. (예: none, underline, line-through)
  • line-height: 줄 간격을 조절합니다. (예: 1.5, 24px)
  • letter-spacing: 글자 사이 간격을 조절합니다. (예: 1px, -0.5px)
  • text-shadow: 글자에 그림자를 넣습니다. (예: 2px 2px 4px gray;)

2. 배경과 박스 마법 (Background & Box Model)

  • background-color: 배경색을 칠합니다. (예: transparent, white, orange)
  • background-image: 배경 사진을 넣습니다. (예: url('image.jpg'))
  • width / height: 가로/세로 크기를 정합니다. (예: 100px, 50%, auto)
  • padding: 테두리와 내용 사이의 안쪽 여백입니다.
  • margin: 테두리 바깥쪽의 바깥 여백입니다. (margin: auto는 가운데 정렬 꿀팁!)
  • border: 테두리 두께, 종류, 색상을 한 번에! (예: 2px solid #333;)
  • border-radius: 모서리를 둥글게 깎습니다. (50%를 주면 원이 돼요!)
  • box-shadow: 박스 전체에 그림자를 넣습니다. (예: 0 4px 10px rgba(0,0,0,0.1);)
  • overflow: 내용이 박스보다 클 때 어떻게 할까요? (예: visible, hidden, scroll)

3. 레이아웃의 제왕 (Flexbox & Grid)

  • display: flex;: 요소를 가로/세로로 자유롭게 배치하는 마법 박스를 만듭니다.
  • flex-direction: 정렬 방향을 정합니다. (예: row(가로), column(세로))
  • justify-content: 주축(가로) 정렬입니다. (예: center, space-between, space-around)
  • align-items: 교차축(세로) 정렬입니다. (예: center, flex-start, flex-end)
  • flex-wrap: 자리가 좁으면 다음 줄로 넘길까요? (예: nowrap, wrap)
  • gap: 요소들 사이의 간격을 일정하게 줍니다. (예: 20px)
  • display: grid;: 바둑판 모양의 격자 레이아웃을 만듭니다.

4. 위치 마법 (Positioning)

  • position: relative;: 원래 있던 자리를 기준으로 조금씩 옮깁니다.
  • position: absolute;: 부모 요소를 기준으로 자유롭게 날아다닙니다.
  • position: fixed;: 화면에 딱 붙어서 스크롤해도 따라옵니다.
  • position: sticky;: 특정 위치에 도달하면 딱 붙습니다.
  • top / bottom / left / right: 옮길 위치를 지정합니다. (예: top: 10px; left: 0;)
  • z-index: 겹쳐진 요소 중 누가 위에 보일지 정합니다. (숫자가 클수록 위!)

5. 특별한 상태 & 가상 요소 (Pseudo)

  • :hover: 마우스를 올렸을 때만 발동합니다.
  • :active: 클릭하는 순간에만 발동합니다.
  • :focus: 입력창(input)을 클릭하여 입력 중일 때 발동합니다.
  • :nth-child(n): 여러 개 중 n번째 요소만 골라냅니다. (예: li:nth-child(2))
  • ::before / ::after: 요소의 앞이나 뒤에 가짜 내용을 만듭니다. (content 속성 필수!)

6. 부드러운 변화와 변신 (Transition & Transform)

  • transition: 수치가 변할 때 부드럽게 보여줍니다. (예: all 0.3s ease-in-out;)
  • transform: scale(): 크기를 키우거나 줄입니다. (예: scale(1.2))
  • transform: rotate(): 회전시킵니다. (예: rotate(45deg))
  • transform: translate(): 위치를 옮깁니다. (예: translate(10px, 20px))
  • opacity: 투명도를 조절합니다. (0은 투명, 1은 선명)
  • cursor: 마우스 포인터 모양을 바꿉니다. (예: pointer, help, crosshair)

7. 단위의 세계 (Units)

  • px: 픽셀. 절대적인 크기입니다.
  • %: 부모 요소의 크기에 비례합니다.
  • rem / em: 글자 크기에 비례합니다. (보통 1rem = 16px)
  • vw / vh: 화면 전체 너비/높이에 비례합니다. (100vw = 전체 너비)
미션 1

배경색 칠하기

웹페이지 전체 배경색을 연한 파란색으로 칠해볼까요?

힌트: body 태그 사용
body { background-color: #f0f8ff; }
미션 2

제목 색상 바꾸기

가장 큰 제목이 눈에 띄도록 진한 파란색으로 바꿔주세요!

힌트: h1 태그 사용
h1 { color: #1e3a8a; }
미션 3

제목 가운데 정렬하기

제목이 예쁘게 가운데에 오도록 위치를 옮겨볼게요.

힌트: text-align 속성 사용
h1 { text-align: center; }
미션 4

소개글 크기 키우기

사람들이 잘 읽을 수 있게 소개글의 크기를 18px로 키워주세요.

힌트: p 태그 사용
p { font-size: 18px; }
미션 5

사진 테두리 둥글게 깎기

테마파크 사진이 부드러워 보이도록 모서리를 둥글게 깎아주세요~

힌트: border-radius 속성 사용
img { border-radius: 20px; }
미션 6

버튼 예쁘게 꾸미기

제출 버튼에 색상과 안쪽 여백(padding)을 줘서 누르고 싶게 만들어요!

힌트: button 태그 사용
button { background-color: #38bdf8; color: white; border: none; padding: 10px 20px; border-radius: 5px; cursor: pointer; }
미션 7

버튼에 호버 마법 걸기

버튼에 마우스를 올렸을 때 진한 색으로 변하는 마법을 걸어주세요!

힌트: button:hover 사용
button:hover { background-color: #0284c7; }
미션 8

섹션 꾸미기 (박스 마법)

내용이 들어있는 상자(content-box)에 예쁜 색상과 테두리를 줘서 특별한 구역으로 만들어볼까요?

힌트: .content-box 클래스 선택자 사용
.content-box { background-color: #e0f2fe; padding: 20px; border-radius: 10px; border: 2px dashed #38bdf8; }
