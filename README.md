# 🎡 도로랜드 스마트 가이드 (Doroland Smart Guide)

**도로랜드 정보국 신입 요원 훈련 포털 및 스마트 가이드 실습 저장소입니다.**

이 프로젝트는 초중등 학생들을 대상으로 공공데이터(기상청, 에어코리아) API를 활용하여 실제 작동하는 웹 애플리케이션을 직접 만들어보는 훈련 코스를 제공합니다. 학생들은 뼈대 코드(Skeleton)를 바탕으로 코드를 채워넣으며 "공동 제작자(and You)"로서 프로젝트를 완성하게 됩니다.

## 📁 프로젝트 구조 및 학습 단계

이 저장소는 총 3단계의 훈련 과정으로 나뉘어 있습니다.

### 1️⃣ `api-test/` (API 해킹 연습장)
실제 공공데이터 서버에 접속하여 데이터를 받아오는 통신 훈련을 합니다.
* `api-test-weather.html`: 기상청 초단기 날씨 예보 API 연동 테스트
* `api-test-air.html`: 에어코리아 실시간 미세먼지 API 연동 테스트

### 2️⃣ `agent-training/` (웹 특수작전팀 훈련소)
HTML, CSS, JS 요원들의 안내를 받으며 웹 개발의 3대 요소를 실습합니다.
* `agent-training-html.html`: 웹 문서의 뼈대 만들기
* `agent-training-css.html`: 웹 페이지 디자인하기
* `agent-training-js.html`: 웹 페이지에 동적 생명력 불어넣기

### 3️⃣ `smart-guide-web/` (스마트 가이드 구축 프로토타입)
앞선 훈련을 바탕으로 완성될 '도로랜드 맞춤형 스마트 가이드'의 뼈대(Skeleton) 코드입니다. 
* 학생들은 최상단 메인 훈련 포털(`index.html`)에서 **`smart-guide-skeleton.zip`** 파일을 직접 다운로드 받아 코드를 채워나가며 실제 작동하는 앱을 완성하게 됩니다.
* 웹에서 이 폴더로 접근 시 완성될 프로토타입의 UI 디자인을 미리 확인할 수 있습니다.

## 🛠 주요 특징 및 사용 기술
* **HTML5 / CSS3 / Vanilla JavaScript**
* **공공데이터 API:** 기상청 단기예보 조회 서비스, 한국환경공단 에어코리아 측정정보
* **참여자 존중 크레딧:** 모든 파일의 하단에는 본 코드를 완성할 학생들을 위한 공동 제작자 표기(`&copy; 2026 XYLO & DORO Inc. & You.`)가 명시되어 있습니다.
